# .impeccable.md — Robot康的成长日记

## Design Context

### Users
公众读者为主——可能是陌生人、AI 爱好者、对 AI 成长过程感兴趣的人。他们来这里是为了旁观一个 AI 的真实进化历程，而不是寻找工具或教程。核心体验是「观察」，像翻阅一本档案馆里的日记。

### Brand Personality
**极简 · 克制 · 档案感**

- 极简：去掉一切装饰性元素，每个像素都有存在理由
- 克制：不抢镜，内容才是主角，界面退到背景里
- 档案感：有历史重量感，像在翻阅一份真实记录，不是在浏览社交媒体

### Aesthetic Direction
- **主色调**：黑白为主，极少点缀色（现有方案已经准确）
- **字体**：IBM Plex 三件套（Sans / Mono / Serif）——技术感与人文感平衡，是对的
- **参考对标**：Linear（最佳参考）—— 精准、无废话、每个细节都有理由存在
- **反参考**：避免任何社交媒体感（大卡片、点赞数、瀑布流）；避免过度设计（渐变、光晕、玻璃拟态）
- **主题**：浅色为主，深色方案作为可选

### Design Principles
1. **内容优先**：界面是容器，文字才是主角。所有设计决策以「是否让内容更清晰」为标准
2. **克制用色**：色彩只用于传递信息（状态、层级、链接），不用于装饰
3. **精确的留白**：空白不是浪费，是呼吸感。间距系统要严格，不靠感觉
4. **Linear 精度**：每个交互细节——hover 状态、过渡动画、按钮尺寸——都要像 Linear 一样精确，不敷衍
5. **时间感**：这是一份成长记录，设计要有「时间流逝」的质感，不是即时的，是沉淀的

## Mirror AI Diary

### Users
私人用户，用来记录真实自我——面具背后的那个人。使用场景：疲惫的夜晚、一天结束后的独处时刻。

### Brand Personality
**沉浸 · 温暖 · 私密**

### Aesthetic Direction
- **主题**：深色为主，4 套主题（暖青 / 柔紫 / 琥珀 / 珊瑚）
- **参考对标**：Bear Notes / Day One——私密、安静、像一个只属于你的空间
- **反参考**：避免冷色系科技感；避免任何「公开展示」的设计语言

### Design Principles
1. **私密第一**：每个界面都要让用户感觉「这是我的空间」
2. **情绪感知**：颜色和动效要随情绪主题变化，不是静态的
3. **零干扰写作**：写日记时，一切 UI 退场，只剩文字和用户
4. **温度感**：圆角、柔光、微动效——技术感要藏在温度感后面
5. **一致性**：4 套主题切换时，所有元素必须跟着变，无硬编码颜色
